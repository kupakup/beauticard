# BeautiCard

Веб-редактор обложек для статей. Два столбца: превью слева, панель настроек справа. Без сборки — React/Babel грузятся с CDN.

## Возможности

- **8 стилей-пресетов**: Журнал, Книга, Модерн, Brutalist, Editorial, Tech, Photo, Gradient — каждый со своими шрифтами, цветами, расположением и фирменными декорациями.
- **4 формата**: 3:4 (журнал), 1:1 (соцсети), 16:9 (блог), 9:16 (stories). Превью авто-подгоняется по экрану.
- **Три текстовых поля**: надпись над заголовком (kicker), заголовок, подпись. У каждого — отдельный шрифт, размер, толщина, цвет, регистр, межбуквенный, высота строки.
- **6 режимов фона**: заливка, градиент с углом, конструктор геометрических блоков, загруженное фото, поиск по Unsplash, паттерн (точки / сетка / линии).
- **История версий**: undo / redo (⌘Z, ⌘⇧Z) и попап с таймлайном до 30 точек.
- **Рандомизатор** (⌘R): один клик — новый стиль и случайный фон.
- **Шаблоны** — 6 готовых сценариев (Дизайн-вестник, Серия эссе, Tech Weekly, Lookbook, Editorial, Punch!).
- **Скачивание PNG** — снимок превью через html-to-image, длинная сторона 1200 px.

## Запуск

Открыть `index.html` в браузере. Для Unsplash и кросс-доменных загрузок нужен локальный сервер:

```bash
python3 -m http.server 8000
# затем http://localhost:8000
```

## Поиск картинок (Unsplash)

1. Зарегистрируй бесплатное приложение на https://unsplash.com/oauth/applications
2. Скопируй **Access Key**
3. Открой вкладку **Фон** → режим **Сток** → вставь ключ → введи запрос. Ключ хранится в `localStorage` браузера.

Используется по [Unsplash API Guidelines](https://unsplash.com/api-terms).

## Стек

- **React 18** + **Babel Standalone** через CDN (JSX без сборки)
- **html-to-image** для PNG-снапшота превью
- **Google Fonts**: Manrope, Space Grotesk, Bricolage Grotesque, Instrument Serif, Playfair Display, EB Garamond, JetBrains Mono, Archivo Black
- CSS-токены и container queries для типографики, которая автоматически масштабируется с обложкой

## Структура

```
beauticard/
├── index.html    # точка входа: подключает CSS, шрифты, React и JSX-файлы
├── styles.css    # дизайн-токены, лейаут, контролы
├── presets.jsx   # 8 пресетов, фоны, декорации, CoverPreview
├── panels.jsx    # правая панель: Стиль / Текст / Типографика / Фон / Шаблоны
├── app.jsx       # шапка, история, рандом, скачивание PNG
└── README.md
```
