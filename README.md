# Beauticard

Простой веб-генератор обложек для постов в мессенджерах и блогах. Один HTML-файл, никакой сборки.

Текст вводится в textarea — карточка-обложка обновляется на лету. Дальше выбираешь фон (градиент / заливка / картинка), настраиваешь шрифт, выбираешь пропорции и скачиваешь PNG или JPG.

## Возможности

- **Фон**: градиент (2–3 цвета, угол, пресеты), сплошная заливка (палитра, hex-ввод, пресеты), картинка (загрузка с диска, drag & drop, URL, поиск по Unsplash).
- **Текст**: 13 Google Fonts с кириллицей, размер (% от меньшей стороны карточки), толщина (Regular/Bold/Black), цвет, выравнивание, положение (сверху/центр/снизу), межстрочный, тень.
- **Картинка**: настройка затемнения (цвет + прозрачность) и размытия фона.
- **Пропорции**: 1:1, 16:9, 9:16, 4:5, 1.91:1 (OG / Telegram).
- **Скачивание**: PNG или JPG в 720 / 1080 / 1920 по длинной стороне.

## Запуск

Самый простой вариант — открыть `index.html` в браузере двойным кликом. Tailwind, html2canvas и Google Fonts подгружаются с CDN.

Для поиска по Unsplash нужен локальный сервер (иначе fetch блокируется на `file://`):

```bash
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

## Поиск картинок (Unsplash)

1. Зарегистрируй бесплатное приложение на https://unsplash.com/oauth/applications
2. Скопируй **Access Key**
3. В настройках картинки → вкладка **Unsplash** → вставь ключ и нажми «Сохранить». Ключ хранится только в `localStorage` твоего браузера.
4. Введи запрос (например, `горы`) → клик по фото — оно становится фоном. Атрибуция автоматически появляется под превью.

Фото с Unsplash используются по их [API Guidelines](https://unsplash.com/api-terms): авторство и ссылки на источник проставляются автоматически.

## Стек

- [Tailwind CSS](https://tailwindcss.com/) (Play CDN)
- [html2canvas](https://html2canvas.hertzen.com/) — снимок DOM в canvas для скачивания
- [Google Fonts](https://fonts.google.com/) — подгрузка шрифтов
- Vanilla JS, без сборки и зависимостей
